## menghitung luas lingkaran
### rumus luas dan keliling lingkaran
Luas     = π × r²
Keliling = 2 x π × r

luas = phi*(r*r)
keliling = 2*phi*r

-nilai phi adalah 3.14
-r adalah jari-jari lingkaran
